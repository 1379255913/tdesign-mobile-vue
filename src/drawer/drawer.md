# Drawer 抽屉

## 组件类型

抽屉从父窗体边缘滑入，覆盖住部分父窗体内容。用户在抽屉内操作时不必离开当前任务，操作完成后，可以平滑地回到到原任务，常用于侧边栏导航

## Props

| 属性            | 类型    | 默认值 | 必传 | 说明                                                                                    |
| --------------- | ------- | ------ | ---- | --------------------------------------------------------------------------------------- |
| v-model/visible | Boolean | false  | Y    | 显示与隐藏                                                                              |
| sidebar         | Object  | -      | N    | 列表参数，包含 name(菜单名称),path(跳转路径),icon(图标支持function、slot、string) |
