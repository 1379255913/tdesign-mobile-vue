## API
### Result Props

| 名称             | 类型          | 默认值 | 说明      | 必传 |
| ---------------- | ------------- | ------ | ------ | ---- |
| title            | String / Slot | -      | 标题     | N    |
| description      | String / Slot | -      | 描述文字       | N    |
| icon             | String        | -      | 图标名称      | N    |
| image            | String / Slot | -      | 图片地址  | N    |
| theme           | String         | default| 内置主题。可选项：default/success/warning/error | N  |
